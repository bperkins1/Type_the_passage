Type the Passage is a typing test. The "test" is a 50 word passage retrieved online at gutenberg.org, where books are publically available.

To use this program, run the main.py file from the command line

Hit begin, type the passage, then hit submit when finished. 
Your speed and accuracy will be displayed
Then, you can start a new test or open the URL that has the book where the passage you typed is from.
